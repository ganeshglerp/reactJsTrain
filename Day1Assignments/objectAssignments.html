ES6 Objects

Objects are the collection of key/value pairs that can be modified throughout the lifecycle of an object. Objects allow us to define custom data types in JavaScript.

Unlike primitive data types, we can represent complex or multiple values using objects. The values can be the array of objects or the scalar values or can be the functions. Data inside objects are unordered, and the values can be of any type.

Object Initializers

Like the primitive types, objects have a literal syntax: curly bracesv {...} along with an optional properties list. The property is a "key:value" pair, where the key is a string or a property name, and the value can be anything. Following is the syntax for defining an object.

                    var student = {
                        studentFirstName:"Ganesh", studentFirstName:"Golla",
						function () { ... }, 
                        address: ["Bangalore","Hospet"] 
                    }

In ES6, assigning a property value that matches a property name, you can omit the property value.

                    var studnetDetaild = {student}

Computed property names, which is a part of object literal syntax, and it uses the square bracket [] notation. It allows us to put an expression within the square brackets [] that will be computed and used as the name of the property

var department = 'GSS';  
					var student = {  
						studentId : 102,  
						studentName : 'Ganesh Golla',  
						[department]:'Production'  
					}

The Object() Constructor

JavaScript provides a special constructor function called Object() to build the object. The new operator is used to create an instance of an object. To create an object, the new operator is followed by the constructor method

                    var student = new Object();
